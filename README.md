# Birko.Workflow.CosmosDB

Azure Cosmos DB workflow instance persistence for the Birko Framework.

## Features

- Workflow instance persistence in Cosmos DB
- Query by state, status, or workflow name
- JSON-serialized workflow data and history

## Dependencies

- Birko.Workflow
- Birko.Data.CosmosDB
- Microsoft.Azure.Cosmos
